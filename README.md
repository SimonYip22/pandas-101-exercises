# Pandas 101 Exercises

This repository contains my solutions to a curated subset of the 101 Pandas Exercises from MachineLearningPlus.

The purpose of this project is not to complete every exercise, but to work through the most useful Pandas questions for my current level. The selected exercises are intended to build practical confidence with Pandas fundamentals.

The exercises are completed in Jupyter Notebook format so that each solution can include code, outputs, explanations, and reflections.
